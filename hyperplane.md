| Space Dimension | Divider Name           | Why?                                   |
| --------------- | ---------------------- | -------------------------------------- |

#### âœ… **Key Rule**

A **hyperplane** is always:

* **One dimension less** than the space it's in.

| Space Dimension | Hyperplane Dimension | Example Equation                 |
| --------------- | -------------------- | -------------------------------- |
| 2D              | 1D                   | `x + y = 3` (Line)               |
| 3D              | 2D                   | `x + y + z = 3` (Plane)          |
| 4D              | 3D                   | `x + y + z + w = 3` (Hyperplane) |

***
